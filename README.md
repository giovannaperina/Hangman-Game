# Hangman Game

The Toy Story Game is a fun Hangman that features dynamically HTML and CSS powered by Vanilla Javascript.

![GameView](assets/images/gameview.png)

### Overview 
The game provides an incredible experience to the user. Including, animations every time the user choose a letter on the alphabet. A related music on background, and different action if the player either win or lose.
